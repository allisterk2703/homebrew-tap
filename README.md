# homebrew-tap

Personal [Homebrew](https://brew.sh) tap for my macOS apps.

## Usage

```sh
brew tap allisterk2703/tap
brew trust allisterk2703/tap   # third-party taps must be trusted once
```

## Available casks

| Cask | Description |
|---|---|
| [`flashcard`](https://github.com/allisterk2703/flashcard) | Simple, fast and elegant flashcards app with spaced repetition |

```sh
brew install --cask flashcard
```
